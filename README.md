★これは何？
　GTOによるKohn-Sham-Roothaan方程式を解くことにより、水素原子の基底状態のエネル
　ギーを求めるコードです。ビルドには以下のライブラリが必要です。
　・Boost C++ Libraries
　・Eigen
　・Libxc
